## Image Tips

- Use **real photos** of your actual work for best trust/conversion
- If no real photos yet, use **stock images** from:
  - Unsplash.com (free)
  - Pexels.com (free)
  - Pixabay.com (free)
- Search terms: "large format printing", "billboard installation", "office wall branding", "roll up banner", "LED sign board", "shop signage India"
- Compress images with TinyPNG.com before uploading
- Keep each image under 200KB for fast loading

## How to Add Images

Once images are placed in the correct folders, the CSS will automatically pick them up.
No code changes needed if you follow the exact filenames above.
